Project Overview

Barangay Management System

The system centralizes resident management, service requests, community events, and reporting into one operational platform. It reduces manual paperwork and helps barangay staff process requests with predictable workflows.

Our implementation includes resident and admin modules, API integrations, payment workflow support, and notification channels designed for real deployment scenarios.
